# Restaurant Reservation App

This is a React web application for booking restaurant tables. Users can select their name, email, date, time, and number of guests to make a reservation. The app validates the form, saves bookings in `localStorage`, and displays them in a table. Accessibility and responsive design are applied where possible.

---

## Features

- Reserve a table with name, email, date, time, and number of guests
- Form validation using **Formik** and **Yup**
- Accessibility with **ARIA attributes** and semantic HTML
- Bookings are saved in **localStorage** and displayed in a table
- Dynamic time selection based on the chosen date
- Unit tests with **Jest** and **React Testing Library**

---

## Technologies Used

- React
- Chakra UI
- Formik + Yup
- Jest + React Testing Library
